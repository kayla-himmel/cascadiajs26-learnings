Tools Worth Knowing

A few tools that came up at CascadiaJS 2026 — worth keeping in your back pocket.

VideoJS

The newly open-sourced, rewritten video player from Mux — framework-agnostic and production-ready.

VideoJS docs (opens in new tab)

From the talk

Mentioned as a drop-in replacement for Mux's old player. Now open source and re-architected from the ground up.

Wasp

A full-stack web framework for React + Node.js that lets you declare your app structure in a simple config language.

Wasp on GitHub (opens in new tab)

From the talk

Briefly mentioned as a batteries-included alternative to assembling your own React/Node stack.

Kiro

Amazon's AI-assisted coding tool — similar to Claude Code but built with AWS integrations in mind.

kiro.dev (opens in new tab)

From the talk

Used in a couple of lightning talks and demos at the conference. Doesn't require AWS, but shines on AWS projects.
