How to use this repo

  1. Clone the repo and run npm install && npm run dev
  2. Open a file in VS Code — each one has inline comments guiding what to change
  3. Save the file — Vite's HMR updates the browser instantly
  4. Run npm run check:refs to validate the documentation linking examples
